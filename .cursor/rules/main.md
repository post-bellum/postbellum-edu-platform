## Project Overview
This is an educational platform for teachers to discover, customize, and use historical lesson materials. The platform features video content, downloadable resources, and collaborative editing capabilities. Target audience: Czech and Slovak teachers.

## Tech Stack
- Next.js 15 with App Router (React 19)
- TypeScript 5.6+ with strict mode
- Tailwind CSS 3.4+ with custom design system
- shadcn/ui for UI components
- Supabase for auth, database, and storage
- TipTap for WYSIWYG editing
- React PDF for PDF generation
- Vimeo for video hosting
